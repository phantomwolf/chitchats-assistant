# Privacy Policy

**Last updated:** March 17, 2026

## Overview

This Chrome extension is designed to assist users in automating actions on specific Chit Chats shipment pages. Your privacy is important, and this extension is built to minimize data collection and ensure your information stays under your control.

---

## Information Collected

This extension only collects information that you explicitly provide. This may include configuration data or input values you choose to save for convenience.

* All data is provided directly by the user
* No automatic data collection occurs
* No browsing history or unrelated site data is collected

---

## How Data Is Stored

* Data is stored locally using Chrome’s extension storage (`chrome.storage`)
* If enabled, data may be synchronized across your Chrome browsers using `chrome.storage.sync`
* Data remains within your browser environment and is not stored on external servers controlled by the developer

---

## Data Usage

The stored data is used solely to:

* Autofill or assist with form interactions on supported Chit Chats pages
* Improve usability and reduce repetitive manual input

No data is used for analytics, tracking, or profiling.

---

## Data Sharing

This extension does **not**:

* Transfer data to any external servers
* Share data with any third parties
* Sell, rent, or disclose user data

All data remains within your browser.

---

## Permissions

The extension may request permissions such as:

* **Storage** — to save user-provided configuration data
* **Host permissions (Chit Chats website only)** — to interact with shipment pages and perform automation

These permissions are used strictly for the extension’s core functionality.

---

## Data Security

Because all data is stored locally within Chrome’s storage system, it is protected by Chrome’s built-in security model. No additional transmission or exposure occurs.

---

## User Control

You have full control over your data:

* You can edit or delete stored data at any time through the extension settings
* You can remove all stored data by uninstalling the extension

---

## Changes to This Policy

This Privacy Policy may be updated from time to time. Any updates will be reflected with a revised “Last updated” date.

---

## Contact

If you have any questions about this Privacy Policy, you can contact the developer via the repository where this extension is hosted.
