# ChitChats Assistant
ChitChats Assistant is a chrome extension which helps ChitChats users efficiently buy shipping labels.

## Features

* Fill in manufacturer info.
* Fill in product declaration info, such as descriptions, HS code, country of origin, etc.
* Automatically select package types based on weight.
* Automatically select postage rates based on priority.
* Automatically download shipping label PDF files.
